# Pagination Attribute

| 参数       | 描述         | 类型   | 可选值 | 默认值            |
| :--------- | :----------- | :----- | :----- | :---------------- |
| pageIndex  | 默认页数     | Number | —      | 1                 |
| pageSize   | 默认每页条数 | Number | —      | 10                |
| pageNumber | 最多显示页数 | Number | —      | 5                 |
| pageSizes  | 每页显示条数 | Array  | —      | [10, 20, 50, 100] |
